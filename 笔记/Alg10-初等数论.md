### 🦉本文提及的数论算法模板

#### 扩展欧几里得算法

```c++
int exgcd(int a, int b, int &x, int &y){
    if(b == 0){
        x = 1, y = 0;
        return a;
    }
    int d = exgcd(b, a%b, y, x);
    y -= (a / b) * x;
    return d;
}
```
